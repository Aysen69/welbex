# welbex

### Инструкция

1. Освободить порты 8080 и 3000

2. Запустить следующие команды:

git clone https://github.com/Aysen69/welbex.git

cd welbex

git submodule init

git submodule update

cp .env.examlpe .env

docker-compose build

docker-compose up

3. Открыть в браузере http://localhost:8080

4. Все :)
